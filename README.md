# LuckPay Loan API

Simple Spring Boot REST API project built for loan account integration.

# Tech Used
- Java
- Spring Boot
- MySQL
- JPA/Hibernate
- OpenFeign

 # Features
- Fetch loan details
- Store response in MySQL
- Return API response
- Clean layered architecture

# API
GET /api/loans/{loanAccountNumber}
Example: http://localhost:8080/api/loans/123456
